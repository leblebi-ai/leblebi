## ZFA Architecture Requirements

### Entity Definition

**AppSettings Entity:**
```bash
zfa entity create -n AppSettings --field "id:String,theme:String,accentColor:String,fontSize:String,defaultModelId:String?,language:String,dateFormat:String,timeFormat:String,firstDayOfWeek:String,timezone:String,notificationsEnabled:bool,notificationSound:bool,quietHoursEnabled:bool,quietHoursStart:String?,quietHoursEnd:String?,createdAt:DateTime,updatedAt:DateTime" --json
```

### Clean Architecture Generation

```bash
zfa generate AppSettings --methods=get,update --data --vpc --state
```

### Implementation Notes

1. **Singleton Pattern**: Use single `AppSettings` instance (id: 'default')
2. **Theme Application**: Apply theme change immediately via `ThemeData`
3. **Cloud Sync**: Sync settings to cloud if user is signed in
