## ZFA Architecture Requirements

### Entity Definition

Uses existing **Chat** and **Message** entities (UC-3.1).

**ExportJob Entity:**
```bash
zfa entity create -n ExportJob --field "id:String,type:String,format:String,chatId:String?,outputId:String?,status:String,filePath:String?,createdAt:DateTime,completedAt:DateTime?" --json
```

### Clean Architecture Generation

```bash
zfa generate ExportJob --methods=get,getList,create --data --vpc --state
```

### Implementation Notes

1. **Markdown Export**: Iterate messages, format with timestamps
2. **PDF Export**: Use `pdf` package to render markdown content
3. **Background Processing**: Run export in isolate for large chats
